# Happy Birthday Message Wall

先生の誕生日メッセージをまとめた、ローカルで開ける1ページHTMLサイトです。

## 開き方

`index.html` をブラウザで開いてください。

## メッセージを更新する場所

`index.html` の下部にある `const messages = [` から始まる配列を書き換えると、表示内容を更新できます。

各メッセージはこの形です。

```js
{
  name: "表示名",
  message: "メッセージ本文"
}
```

送信日時やアイコンは使いません。同じ名前が複数あっても、そのまま追加してください。
